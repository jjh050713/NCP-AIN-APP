# 📱 iPhone만 있을 때 설치 방법 (Mac 불필요)

## 가장 간단한 방법: 홈 화면에 추가 (30초)

이 앱은 **PWA(웹앱)** 로 제공됩니다. App Store 없이 iPhone에 설치할 수 있습니다.

### ① Safari로 접속

아래 주소를 **iPhone Safari**에서 엽니다:

```
https://jjh050713.github.io/NCP-AIN-APP/
```

> ⚠️ **Chrome이 아닌 Safari**를 사용해야 합니다.

### ② 홈 화면에 추가

1. Safari 하단 **공유 버튼** (□↑) 탭
2. 아래로 스크롤 → **"홈 화면에 추가"** 탭
3. 이름 **NCP-AIN** 확인 → **추가** 탭

### ③ 완료!

홈 화면에 앱 아이콘이 생깁니다. 탭하면 **전체 화면**으로 앱처럼 실행됩니다.

## 자주 묻는 질문

**Q. App Store에서 설치할 수 없나요?**
A. iOS 네이티브 앱은 Mac + Apple Developer 계정($99/년)이 필요합니다. PWA가 Mac 없이 설치하는 가장 쉬운 방법입니다.

**Q. 인터넷 없이 쓸 수 있나요?**
A. 한 번 열어두면 오프라인에서도 동작합니다.

**Q. 데이터는 어디에 저장되나요?**
A. iPhone Safari 로컬 저장소에 저장됩니다 (북마크, 진행률).

**Q. 주소가 안 열려요**
A. GitHub Pages 배포 후 1~2분 기다린 뒤 다시 시도하세요. PR이 머지되어야 배포됩니다.
